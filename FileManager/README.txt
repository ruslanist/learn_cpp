В будущем, эта программа позволит управлять файлами(удалять, создавать, просматривать контент) как на локальном компьютере, так и на удаленных узлах
Сейчас она на вход принимает файл, который нужно найти на локальном компьютере и выдает YES, если такой файл найден.

Для сборки программы используется {cmake}
Что бы собрать программу, нужно выполнить следующие комманды из директории с этим проектом

mkdir build
cd build
cmake ..
make
./FileManager имя_файла_для_поиска


cmake для сборки программы использует директивы из файла CMakeLists.txt
